branchs


branching stretagys


check which branch
git branch

new branch and cd to it
git checkout -B <name>  // git branch <name>
ex:-  git checkout -B dev // git branch dev

switch branch 
git switch <name>  ex:- git switch dev
example:
1.create a txt file called this_is_from_dev.txt
2.git status  check status of branch 
3.git add this_is_from_dev.txt
4 git commit -m "added dev file"
5.git log // git log --oneline 
(to exit from git log press " q ")
